# microG Services - Fork

microG Servicesは、Google Play Services向けに設計されたアプリケーションを、Play Servicesが利用できないシステム上（Huaweiスマートフォンなど）で実行可能にするFLOSS（フリー／リブレ・オープンソースソフトウェア）フレームワークです。

microG Services - Fork（microG 開発者サービス）は microG のフォークです。Play Store や Play Services のベースバージョンの引き上げ、日本語訳を積極的に行っているプロジェクトです。

### ダウンロードと手順については、[microG 公式wiki](https://github.com/microg/GmsCore/wiki)をご参照ください。
