# microG Services - Fork

microG Services is a FLOSS (Free/Libre Open Source Software) framework to allow applications designed for Google Play Services to run on systems, where Play Services is not available.

microG Services - Fork (microG Developer Services) is a fork of microG. It is a project actively working on upgrading the base versions of the Play Store and Play Services, as well as providing Japanese translations.

### For download and installation instructions, please refer to the [microG official wiki](https://github.com/microg/GmsCore/wiki).
